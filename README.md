# NimLab

**Interactive Game Theory & Reinforcement Learning Playground**

[▶ Live Demo](https://nimprojectweb.vercel.app/nim/) | [📚 Learn the Strategy](https://nimprojectweb.vercel.app/nim/learn.html) | [💻 Source Code](https://github.com/KruthikKoundinyas/nimprojectweb)

---

An educational web application that demonstrates how mathematical optimization (XOR strategy) and reinforcement learning (Q-Learning) solve the same strategic decision-making problem.

Play against 4 AI strategies. Watch the Algorithm Inspector explain every move. Learn why analytical and learned intelligence converge on the same decisions.
