NIM Misère Game – Web-Based AI Strategy Game
This is my first and original web application: a browser-based Misère variant of the classic Nim game, featuring a sleek retro gaming design and an intelligent AI opponent powered by Q-learning.

Key highlights:

Interactive UI: Built with HTML, CSS, JavaScript, and jQuery for a dynamic and responsive user experience.

AI Integration: Implements a self-learning opponent using Q-learning, offering challenging and adaptive gameplay.

Retro-Inspired Design: Styled with a nostalgic aesthetic to create an engaging and visually appealing interface.

Version Control: Managed using Git and GitHub, showcasing collaboration-ready development practices.

This project demonstrates my foundational skills in full-stack web development, machine learning integration, and UI/UX design—all wrapped into a fun and strategic game experience.
